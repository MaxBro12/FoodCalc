# Auth Service

Сервис создания и управления учетными записями пользователей, создания токенов доступа и обновления. Для создания токенов используется своя реализация SimpleJWT, её реализация находится в пакете core. Это тестовая версия и создана для демонстрации возможностей, на продакшене используется продвинутая реализация.
